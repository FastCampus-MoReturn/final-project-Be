# 💰Mori

부동산을 공부한 적은 없지만, 매매 또는 임대해야하는 사람들에게 쉽게 특정 부동산 관련 정보 해설<br>
시세, 대출 현황, 계약 전 추가로 체크해야 할 사항 등 정보를 제공하는 애플리케이션
